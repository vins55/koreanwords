문법

Grammar

1. A -다고하다, V-ㄴ다고/는다고하다, N(이)라고하다
다른 사람의 말이나 내용을 전달할 때 사용하는 간접화법이다.

2. V-아야겠다/어야겠다
화자의 의지, 결심 또는 필요성을 나타낼 때 사용하는 표현이다.

3. A-다고들었다,V-ㄴ다고/는다고 들었다, N(이)라고들었다
다른 사람이나 매체를 동해 들은 내용을 옮겨 전달할 때 사용한다

4. A-대(요), V-대/ 는대(요), N(이)래(요)
'-다고 해요'를 줄여서 말하는 구어체 표현이다. 친한 사람과 이야기할 때 많이 사용한다.

문장

Example Sentences
