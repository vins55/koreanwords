문법

Grammar

1. A-(으)ㄴ가 보다, V-나 보다, N인가 보다
눈에 보이는 상황이나 들은 정보를 바탕으로 추측할 때 사용한다.

2. N(이)나
최선은 아니지만 괜찮은 정도의 차선책을 가볍게 제안할 때 사용한다.

3. V-아/어 보니(까)
어떤 행동을 직접 해 본 후에 알게 된 사실이나 느낌을 말할 때 사용한다.

4. A/V-던데(요)
과거를 회상하며 경험한 사실에 대해 표현하고, 듣는 사람의 반응을 기대할 때 사용한다.
